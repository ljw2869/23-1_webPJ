웹문서 글꼴 색상 등 결정하는 것이 스타일시트 역할- 내용과 디자인을 분류 
디자인적인 요소를 수정하고 싶을 때 <style>태그 부분만 수정하면 됨
웹페이지에 접근하는 기기들이 다양 화면크기에 맞게 배치를 다르게 하려면 CSS를 사용해야 함
스타일: 각각 하나, 스타일시트: 여러 개의 스타일을 한군데에 모아둔 것
문서 안에 만드는 내부스타일시트/ 여러 스타일 시트를 묶어서 따로 밖에 모아둔 것을 외부 스타일시트라고 함
외부스타일시트의 예시: <link rel="stylesheet" href="css/register.css">이렇게 외부의 것을 가져오는 코드 
내부스타일시트를 정의하기 위해 <head>부분에 <style>태그를 사용해서 여기서부터 스타일을 정하는 부분임을 알려줌
주석처리: 영역 선택후 ctrl+?키 
아무것도 스타일이 정해지지 않은 상태를 브라우저 기본스타일이 적용되어 있음
인라인 스타일: 각각 태그에 덧붙여서 정보를 추가하는 방식 
예시: <h1 style="color: blue">라고 색깔을 바로 설정할 수 있음
내부 스타일시트: 문서 안에 스타일 시트를 모아놓은 것 
예시: h1{ //이때 h1을 선택자라고 하고 (뒤에 오는 속성이 어디에 적용될 것인지 알려줌)
color: blue; //속성: 속성값이라고 한다. 
font-size:3.4em; //속성은 여러개 올 수 있다. 
font-style:italic; //이를(속성+속성값) 스타일 규칙이라고 한다.
}
외부 스타일 시트: 스타일 정보만 따로 모아서 별도의 문서를 만들어 웹문서와 연결하는 것
css파일에는 <style>태그가 들어가지 않음
css파일을 연결할 때는 <link href="파일이름" rel="stylesheet"> rel은 관계를 나타내서 연결하는 파일이 스타일시트라고 알려줌 
외부 스타일시트를 사용하는 이유는 여러 웹문서에서도 적용할 수 있기 때문
외부에서 연결된 상태에서도 <style>태그로 내부에서 바꿔서 쓸 수 있다 

전체 선택자: 웹문서 전체에 적용되는 선택자
*기호 이용 브라우저 기본스타일을 바꾸고 싶을 때 주로 사용
예시: 
*{
margin:0;
padding:0;
}
타입선택자: 특정 태그에만 적용되고 싶을 때 
p{: p태그가 적용된 곳에 같은 속성이 적용된다.
}
p태그가 있는 곳에 유독 하나만 다른 설정을 하고 싶을 때 class선택자와 id선택자를 쓸 수 있다. -특정부분만 다르게 설정하고 싶을 때!!
class선택자는 문서안에서 여러번 사용할 수 있음, id선택자는 한번만 사용할 수 있음, 고유성을 가지고 있음
class선택자는 앞에 .를 붙임
그리고 적용할 태그 안에 class="redtext"라고 더 넣어주면 된다. 
<p>태그 안에 특정 글자만 class선택자를 적용하고 싶을 때는 그 글자 밖에 <span>으로 감싸주고 <span class="redtext">라고 설정해주면 된다. 
id선택자는 앞에 #기호를 넣어주면 됨
예시 #container{
border:1px solid #222;
padding:10px;
}
그리고 적용하고 싶은 태그 안에 id="태그이름" 만 넣어주면 됨 
그룹선택자: 같은 스타일 규칙을 가진 여러 개의 선택자를 나열해서 표시
선택자를 ,를 선택해서 구분하여 나열할 수 있다
예시: h1,p{
}
캐스케이딩 스타일시트: 위에서 아래로 흐른다는 뜻, 계단식으로 적용된다는 의미,
선택자에 여러 스타일이 적용될 때 스타일 충돌을 막기 위해서 우선순위에 따라 적용할 스타일을 결정함
스타일 우선순위는 중요도와 적용범위에 따라 결정이 되고 그 순서대로 적용이 된다. 
태그들의 포함관계에 따라 부모요소의 스타일을 자식요소로 상속되기도 한다. 
스타일 우선순위: 사용자 스타일(시스템에서 만든 스타일로 바꿀 수 없음)-제작자 스타일(css로 만든 스타일)-브라우저 스타일(브라우저 기본스타일)
얼마나 한정지을 수 있는가에 따라: !important -인라인 스타일-id스타일-클래스 스타일- 타입 스타일
소스 순서에 따라: 소스에서 나중에 온 스타일이 먼저 온 스타일을 덮어 씀
스타일 상속: 자식 요소에서 별도로 스타일을 지정하지 않으면 부모 요소에 있는 스타일 속성들이 자식요소로 전달됨.
단, 모든 속성이 상속되는 것은 아니다. 
예시: p태그에 처음에는 black으로, 나중에 p태그에 green으로 스타일을 설정하였다면 결과적으로 초록색으로 바뀌어져 있음
인라인 스타일이 위에서 <style>로 변경해준 것 보다 우선순위에 있다.
