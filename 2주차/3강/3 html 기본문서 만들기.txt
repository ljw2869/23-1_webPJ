<!doctype html> 이 코드는 무조건 첫번째 줄에 있어야 하며 현재 문서가 html5언어로 작성한 웹문서라는 뜻이다.
<html>~</html>이 코드는 웹문서의 시작과 끝을 나타내는 태그로, 두번째 줄과 마지막 줄에 있어야 하는 코드이다. 
index.html 에 있는 lang="ko"라는 것은 이 웹문서가 한글로 작성되었다는 뜻
<head>~</head>는 웹브라우저가 웹문서를 해석하는 데 필요한 정보를 입력하는 부분. 
<body>~</body>는 실제 웹브라우저 화면에서 나타나는 내용.
head는 정보, body는 내용- head는 웹브라우저에 보이지 않음 문서에 필요한 여러 파일, 외부 파일 링크할 때 넣게 됨

head 에 들어가는 <meta>태그=문서 정보
char 세트는 UTF-8로 설정해 두어야 한글을 화면에 보여줄 수 있음
<meta charset="UTF-8">
head안에는 title태그를 이용해서 문서 제목을 나타낼 수 있음 (tab부분에 들어갈 내용)
<title>HTML 기본문서 </title>
vscode에서 .html 확장자를 입력하여 파일을 만들어 둔 상태라면 !+tab or !+Enter키를 누르면 기본 구조를 완성해줌(다만 언어를 en으로 되어 있으니 ko로 바꿔준다)
웹브라우저 상에서는 들여쓰기 상관없음-포함관계를 쉽게 알 수 있음
